# React starter
this project includes:
- [x] HMR
- [x] React
- [x] Redux
- [x] Material ui
- [x] Graphql
- [x] Docker-compose file
- [x] Babel for es6/es7
- [x] Webpack
- [x] Express
- [x] Node

## Installation
```
git clone git@github.com:Raat/boilerplate.git
npm i
npm run storybook
```

### start express server with webpack-dev-middleware
```
npm run start
```

### Dockerfile is included!!

Run docker compose
```
docker-compose up
```

### run tests
```
npm run mocha
```
